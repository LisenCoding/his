## 功能概览
医院业务处理系统包括挂号医生、门诊医生、检查医生、检验医生、处置医生、药房医生、管理员六个模块，由于模块众多所以选择几个主要功能模块展示出来
>:star:挂号医生：挂号、退号、收费、退费、费用记录查询等
>
>:star:门诊医生：问诊、叫号、创建病例、检查申请、发送处理意见、开具药方等
>
>:star:检查医生：叫号、安排检查、选择检查项目、录入检查结果
>
>:star:药房医生：发药、退药、药库管理等
